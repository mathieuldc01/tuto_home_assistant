    Requirements & Conseils d'installation

    • Pour l'installation de Home Assistant

      • ▶ Git est nécessaire pour cloner le dépôt

        Installation de Git

        • Windows : télécharger depuis le site officiel
        • Linux :
          sudo apt update
          sudo apt install git
        • macOS :
          brew install git
      • ▶ Docker est nécessaire pour exécuter Home Assistant

        Installation de Docker

        • Windows / Mac : Télécharger Docker Desktop
        • Linux :
          sudo apt update
          sudo apt install docker.io docker-compose

        Après installation, vérifier que Docker fonctionne :

        docker --version
        docker compose version

    • Pour le déploiement de capteurs ESP32

      Pour pouvoir flasher l’ESP32, vous devez avoir :

      • ▶ Python version 3.13

        • Téléchargez Python depuis le site officiel : https://www.python.org/downloads/

        Windows

        1. Téléchargez l’installateur
        2. Lancez l’installation
        3. Cochez "Add Python to PATH"
        4. Cliquez sur Install Now
        ⚠️ Très important : cocher Add Python to PATH sinon Python ne sera pas accessible dans le terminal.

        Linux

        Installer Python et pip :

        sudo apt update
        sudo apt install python3 python3-pip

        macOS

        Avec Homebrew :

        brew install python
      • ▶ ESPHome version 2025.12

        • Une fois Python installé, ouvrez un terminal ou une invite de commande et tapez :

        pip install esphome==2025.12.0

        Vérifier l'installation :

        esphome version

        Si la commande ne fonctionne pas :

        python -m esphome version

    • Matériel nécessaire

      • Capteur de température adressable (idéal DS18B20)
      • Résistance 4.7 kohm
      • Raspberry Pi 3B+ ou version d'après
      • ESP32 (dans notre cas ESP-32-C3)
