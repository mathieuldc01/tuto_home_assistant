    9. Installation d'une distribution Debian sur un Raspberry Pi

    Tout d'abord, aller sur le site https://www.raspberrypi.com/software/ et installer le **Raspberry Pi Imager**.
    Une fois le logiciel installé, sélectionner le bon appareil Raspberry Pi :
    ![Capture raspberry pi imager 1](raspberry_pi_imager1.png) Choisir la bonne distribution voulue (ici Debian en x64) :
    ![Capture raspberry pi imager é](raspberry_pi_imager2.png) Sélectionner ensuite la carte micro sd sur laquelle on souhaite installer l'OS. Il ne reste plus qu'à entrer les paramètres voulus dans le raspberry (nom d'utilisateur, mot de passe, paramètres réseaux, ...), puis l'installation peut être lancée.
    Une fois l'installation terminée, insérer la carte micro sd dans le Raspberry Pi puis le démarrer.

    Connexion au raspberry pi

    Ouvrir un invite de commande (cmd) sur un ordinateur connecté au même réseau que le Raspberry Pi : win + r -> *cmd* -> entrée
    Connexion au rapsberry pi :
    ssh NOMUTILISATEUR@NOMRASPBERRY -> MOTDEPASSE
    Il faut ensuite télécharger le conteneur dans lequel se trouve home assistant déjà paramétré :
    git clone https://gitlab.ec-lyon.fr/durandu/temperatures-monitoring.git
    Il est ensuite nécessaire de construire le conteneur :
    cd temperatures-monitoring
    docker build -t AJOUTERLENOM
    Il est ensuite possible de lancer le docker :
    sudo docker compose up -d
    Attention, il faut se placer dans le dossier pour pouvoir lancer le conteneur, si ce n'est pas le cas, entrer :
    cd temperatures-monitoring

    Connexion à home assistant

    Pour se connecter à home assistant, il faut tout d'abord récupérer l'ip du Raspberry Pi :
    hostname -I
    Il est ensuite possible de se connecter à homeassistant en entrant le lien suivant dans un navigateur :
    ADRESSEIP:8123

    Connexion à un nouveau réseau

    Lister les réseaux disponibles :
    nmcli device wifi list
    Ajouter la connexion à un réseau :
    sudo nmcli device wifi connect "NOM DU RESEAU" password "MOT DE PASSE"

    Ajout de capteur dans home assistant

    Aller dans Paramètres -> Appareils et services -> 1Wire
    Cliquer sur Ajouter un pont -> remplacer *localhost* par l'adresse IP du raspberry pi et choisir le port 4304 puis valider.
    Si l'ajout ne se fait par correctement, suivre les étapes suivantes.

    Vérification de la bonne connexion des capteurs 1Wire

    Pour vérifier que le protocole 1Wire est bien activé :
    lsmod | grep w1
    Si rien ne s'affiche, ouvrir le fichier de configuration en entrant la commande :
    sudo nano /boot/config.txt
    Ajouter la ligne suivante :
    dtoverlay=w1-gpio
    Enregistrer le fichier puis redémarrer le Raspberry Pi :
    sudo reboot
    Pour lister les capteurs connectés :
    cd /sys/bus/w1/devices/
    ls

    Vérification de la bonne installation de OWServer

    Enter dans le cmd :
    sudo apt-get update
    sudo apt-get install owserver
    Ouvrir le fichier de configuration de OWServer :
    sudo nano /etc/owfs.conf
    Ajouter tout en bas la commande :
    address = 0.0.0.0
    Relancer OWServer :
    sudo systemctl restart owserver

    Ajout d'un capteur à la carte

    Pour associer un capteur à un champ de température de la carte, il suffit d'ajouter le code suivant dans la partie *rules*, en modifiant le nom du capteur et de l'id de salle associé.
    - name: Temperature Salle XXXX entities: - sensor.NOMDUCAPTEUR element: NOMDEL'ELEMENT state_action: - service: floorplan.text_set service_data: >- ${(entity.state !== undefined) ? Math.round(entity.state * 10) / 10 + "°C" : "??°C"} - service: floorplan.class_set service_data: class: static-temp Il faut également ajouter le capteur dans la liste des *entities*.

    Ajout d'une nouvelle carte sur le rapsberry pi

    Pour envoyer un fichier sur le raspberry pi, entrer la commande : scp "CHEMINDUFICHIER" lesframboises@IPDURASPBERRY:/home/lesframboises/ Ensuite, ajouter la carte dans le bon dossier : ssh lesframboises@LesFramboises sudo mv ~/NOMDUFICHIER /home/lesframboises/temperatures-monitoring/config/www/floorplan/
