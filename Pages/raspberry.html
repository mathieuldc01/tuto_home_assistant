    Préparation de la Raspberry Pi

    • Installation de Debian sur le Raspberry Pi

      Télécharger le logiciel officiel : Raspberry Pi Imager

      Une fois le logiciel installé :

      1. Sélectionner le modèle de Raspberry Pi
      2. Choisir le système d'exploitation (Debian x64)
      3. Sélectionner la carte micro SD
      4. Configurer les paramètres (utilisateur, mot de passe, réseau)
      5. Lancer l'installation

      Une fois l'installation terminée, insérer la carte micro-SD dans le Raspberry Pi puis démarrer l'appareil.

    • 1er connexion au Raspberry Pi

      Récupérer l'adresse IP du Raspberry Pi :

      hostname -I

      Dans un navigateur web, entrer :

      ADRESSE_IP:8123

      Vous accéderez alors à l'interface de Home Assistant.

    • Connecter le Raspberry Pi au Wifi

      Lister les réseaux disponibles :

      nmcli device wifi list

      Se connecter à un réseau :

      sudo nmcli device wifi connect "NOM_DU_RESEAU" password "MOT_DE_PASSE"
