Accueil Requirement Home Assistant Raspberry Pi Capteur Plan

  • Cloner le repository

    Clone le projet avec la commande suivante :

    git clone --depth=1 https://gitlab.ec-lyon.fr/durandu/temperatures-monitoring.git

    Puis entre dans le dossier :

    cd temperatures-monitoring

  • Lancer Home Assistant

    Démarre les services :

    docker compose up -d

    Accède à l’interface (depuis un navigateur):

    http://localhost:8123

    Puis suis les instructions pour configurer ton Smart Home.

  • Configurer le dashboard

    (Uniquement si nouveaux capteurs)

    Dans Home Assistant :

    • Aller dans le dashboard W1bis
    • Cliquer sur Edit (icône crayon)
    • Puis "..." → Raw configuration editor

    Copier le contenu du fichier :

    make_config/plan.yaml

    Tu peux copier directement cette configuration :

            title: Floormap views
        views:
        - type: custom:floorplan-card
            full_height: true
            config:
            console_log_level: info
            log_level: info

            defaults:
                hover_action: hover-info
                tap_action: more-info

            image: /local/floorplan/w1bis/plan.svg

            rules:
                - elements: [background_111]
                entity: input_number.test1
                tap_action:
                    action: more-info
                    entity: input_number.test1

    • Puis sauvegarder.

  • Configurer les capteurs

    Édite le fichier suivant :

    make_config/sensor_svg_corresp.yaml

    Format :

    - sensor_id: txt_elem: id_texte_svg bg_elem: id_background_svg

    Génère ensuite la configuration :

    python make_config

    Le fichier généré :

    make_config/plan.yaml

    Copie ce contenu dans le Raw configuration editor du dashboard.
